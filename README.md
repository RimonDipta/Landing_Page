# Ostad MERN Assignment 1: Landing Page

This repository contains the first assignment for the Ostad MERN Stack Learning Course. The objective was to convert a Figma design into a responsive landing page using HTML and Tailwind CSS.

## Features

- **Fully Responsive Layout**: Adapts seamlessly to Desktop, Tablet, and Mobile screens.
- **Pixel-Perfect Implementation**: Closely matches the provided Figma design.
- **Interactive Elements**:
  - Custom hover effects on buttons.
  - Active navigation links highlight on scroll.
- **Custom UI Components**:
  - Gradient border on the email input field.
  - Standardized testimonial cards with equal height and alignment.
  - Floating animations in the Hero section.

## Technologies Used

- **HTML5**: Semantic structure.
- **Tailwind CSS (v4)**: For styling and responsiveness.
- **Google Fonts**: Poppins, Roboto, Abel, DM Sans.
- **FontAwesome**: Icons.

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
